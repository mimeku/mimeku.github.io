mujoco开发环境

可能用到的命令

  1. 下载文件： wget url
  2. 拷贝： cp source_path to_path
  3. 服务器远程传输（ssh）：scp source_path user@address:to_path
  4. 修改文件夹名： sudo mv former_name/ new_name/

Mujoco

安装

Mujoco下载目录

mujoco200 linux 下载地址：https://www.roboti.us/download/mujoco200_linux.zip

创建~/.mujoco文件夹

将下载好的mujoco200_linux复制到~/.mujoco文件夹中

解压mujoco200_linux.zip

将mujoco200_linux重命名为mujoco200，一定要进行重命名，否则之后母鸡哦凑的配置中搜索不到该文件夹

将mjkey.txt复制到~/.mujoco和~/.mujoco/mujoco200/bin两个文件夹中
