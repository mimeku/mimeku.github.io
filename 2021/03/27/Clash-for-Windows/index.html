Clash for Windows

在windows上其实有一个和ubuntu很大的问题就是，通过SSR或V2ray设置的代理，只能给软件层设置代理，而终端是没有代理的。

但是windows下往往也同样要用git clone， pip等命令，没有代理就只能换源等，非常痛苦

曾经使用过 各类博客 等推荐的方法，但是都很不方便

直到知名博主，AI方向前沿研究人员，元强化学习领军者 @typoverflow 的推荐，知晓了最适合windows的代理 Clash

Clash各种详细教程 这篇博客介绍的应该足够详细

SS订阅转Clash 直达链接

最好用的windows代理

为什么说Clash是最好用的windows代理呢？

  1. UI实在是太友好了，各种功能清晰可见，相比于SSR/V2ray，面向小白的主要功能都通过直观的按钮等呈现；比如配置PAC的各种规则
  2. 提供的 直接打开配置了代理的终端 的功能
